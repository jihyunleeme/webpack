### 새 원격저장소로 연결하기
1. 로컬 저장소 `.git` 파일 삭제  
    ```bash 
    rm -rf .git
    ```
2. git init 로컬 저장소 초기화
    ```git
    git init
    ```
3. 파일 추가 및 commit

4. 원격 저장소 연결 및 확인  
    ```git 
    git remote add origin [url]
    git branch -M main
    ```

5. 저장소 연결 후 push!

- 기타 이외 명령어
```bash
git branch -a // 모든 브랜치 목록확인
git remote -v // remote 확인하기
git remote remove [repositoryName] // remote 연결 지우기 
```
